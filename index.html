What is ExonJS?

ExonJS is a poorly-made JavaScript framework designed to make your brain explode. It offers features such as variable-change listening, random utility functions, percent-to-pixel conversion, and custom styling with ExonStyle.

Core Features

Listening to Variable Changes

Track and respond to changes in variables.


const exon = new ExonJS();
let counter = exon.exonVar(0, (oldValue, newValue) => {
    console.log(\`Counter changed from \${oldValue} to \${newValue}!\`);
});

Random Functions

Utilize random functions for array elements and number ranges.


let randomElement = exon.randChoice([1, 2, 3, 4, 5]);
let randomNumber = exon.randMinMax(1, 100);

Percent to Pixels Conversion

Convert percentages to pixel values based on document size.


let halfHeight = exon.percentToPixelsHeight(50);
let halfWidth = exon.percentToPixelsWidth(50);

Custom Styling with ExonStyle

Apply custom styles directly within JavaScript.


exon.Style('myStyle', JSON.stringify({
    'font.size': 20,
    'back.color': '#ffffff',
    'pos.set': 'center'
}));
