ExonJS Documentation

Comprehensive Guide to Using ExonJS

Overview

ExonJS is a JavaScript framework designed to simplify web development. It offers functionality for dynamic styling, UI element creation, random utilities, and reactive variables.

Class: ExonJS

The main class of the ExonJS framework, providing various methods for creating and managing UI elements, applying styles, and handling random functions.

Constructor: ExonJS()

Initializes a new ExonJS instance.

Method: Style(name, json)

Defines a style that can be applied to UI elements.


exon.Style('myStyle', JSON.stringify({
    'font.size': 20,
    'back.color': '#333',
    'size.width': 150,
    'size.height': 50,
    'font.family': 'Arial, sans-serif',
    'pos.left': 100,
    'pos.top': 50
}));

Method: Button(txt, func, exonProps)

Creates a button element with the specified text, click function, and properties.


const btn = exon.Button('Click Me', () => alert('Button Clicked!'), { 'style': 'myButtonStyle' });

Method: Text(txt, exonProps)

Creates a text element (anchor tag) with the specified text and properties.


const text = exon.Text('Hello World!', { 'style': 'myTextStyle' });

Method: getElementById(id)

Returns the EXON element with the specified ID.


const elem = exon.getElementById('myElementId');

Method: nline()

Inserts a line break in the document body.

Method: percentToPixelsWidth(percent)

Converts a percentage to pixels based on the document width.


let pixels = exon.percentToPixelsWidth(50); // Converts 50% to pixels

Method: percentToPixelsHeight(percent)

Converts a percentage to pixels based on the document height.


let pixels = exon.percentToPixelsHeight(50); // Converts 50% to pixels

Method: render(elem)

Renders a UI element to the document body with applied styles and an auto-generated ID.


const header = exon.Text('Welcome!', { 'style': 'headerStyle' });
exon.render(header);

Method: randMax(max)

Generates a random integer from 0 to the specified maximum value.


let random = exon.randMax(10); // Generates a number between 0 and 10

Method: randMinMax(min, max)

Generates a random integer between the specified minimum and maximum values.


let random = exon.randMinMax(5, 15); // Generates a number between 5 and 15

Method: randChoice(array)

Selects a random element from an array.


let choice = exon.randChoice([1, 2, 3, 4, 5]); // Selects a random element from the array

Method: randChoiceDict(dict)

Selects a random key from an dict.


let choice = exon.randChoice({a1: true, a2: true, a3: false}); // Selects a random key from the array

Method: exonVar(initv, func)

Creates a reactive variable that triggers a function when its value changes.


let counter = exon.exonVar(0, (oldValue, newValue) => {
    console.log(\`Counter changed from \${oldValue} to \${newValue}!\`);
});

Style Syntax

Styles are defined using JSON objects with the following properties:

  • back.color: Sets the background color of the element (e.g., #f00).
  • size.width: Sets the width of the element in pixels (e.g., 200).
  • size.height: Sets the height of the element in pixels (e.g., 100).
  • font.size: Sets the font size in pixels (e.g., 16).
  • font.family: Sets the font family (e.g., 'Arial').
  • pos.left, pos.right, pos.top, pos.bottom: Sets the position of the element (e.g., 50 for 50 pixels from the left).
  • pos.set: Currently only supports 'center' for centering the element.

Examples

Basic Button Example


const button = exon.Button('Click Me', () => alert('Clicked!'), { 'style': 'buttonStyle' });
exon.render(button);

Styling Example


exon.Style('buttonStyle', JSON.stringify({
    'font.size': 16,
    'back.color': '#007bff',
    'size.width': 150,
    'size.height': 50,
    'font.family': 'Arial, sans-serif',
    'pos.set': 'center'
}));
const styledButton = exon.Button('Styled Button', () => alert('Styled!'), { 'style': 'buttonStyle' });
exon.render(styledButton);
